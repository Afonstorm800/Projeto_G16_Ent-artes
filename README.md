## Descrição do Projeto
Este projeto consiste numa aplicação web desenvolvida para
apoiar a gestão e operação de uma **Escola de dança**.

A pagina web foi construída com base nos requisitos levantados junto do
cliente ao longo do semestre e evoluiu de mockups para uma implementação
funcional, com persistência local
e uma arquitetura organizada por camadas.
